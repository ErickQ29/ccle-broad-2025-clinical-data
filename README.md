# ccle-broad-2025-clinical-data
Clinical data from the Cancer Cell Line Encyclopedia (CCLE) Broad Institute 2025 release. This repository contains processed clinical annotations, sample metadata, and associated patient/tumour information for cancer cell line models.

## 📋 Current contents

- **`data.tsv`**: Main table containing ~1,000+ cell lines and their clinical annotations (age, cancer type, ploidy, altered genomic fraction, TMB, etc.).
- Main columns: `Study ID`, `Patient ID`, `Sample ID`, `Cancer Type`, `Cell Line Name`, `DepMap ID`, `Age at Sampling`, `Sex`, `TMB`, `Whole Genome Doubling`, etc.
